# 2026 N서울타워 글로벌 나잇워크 (2026 N Seoul Tower Global Night Walk)

| 항목 | 내용 |
|---|---|
| 행사명(국문) | 2026 N서울타워 글로벌 나잇워크 |
| 행사명(영문) | 2026 N Seoul Tower Global Night Walk |
| 기간 | 2026년 6월 13일 ~ 7월 11일 (운영일: 6/13(토), 7/4(토), 7/11(토) — 각 17:30 / 18:30 / 19:30 회차) |
| 장소 | 집결지: 백범광장(Baekbeom Square) / 도착지: N서울타워 팔각정 광장 |
| 가까운 지하철역 | **4호선 회현역 4번 출구, 도보 약 500m** (공식 페이지 명시) |
| 입장료 | KRW 60,000 |
| 외국인 접근성 | **외국어 통역 제공(Foreign language interpretation available)**, 웹사이트 한/영 운영, 장애인 편의시설(휠체어 대여 등) 안내 명시 |
| 한 줄 소개 | 남산을 걷고, K-푸드 마켓·애프터파티까지 즐기는 글로벌 나이트 워킹 체험 (CJ푸드빌 협업) |
| 출처 | [Visit Seoul 공식 영문 페이지](https://english.visitseoul.net/eventsx/2026-N---Global-Night-Walk/ENP8s9qmk) |
| 확인 일시 | 2026-06-28 |

> 비고: 6개 항목(기간·장소·지하철역·요금·외국어지원) 모두 Visit Seoul 공식 페이지에 명시되어 있어 출처 신뢰도가 가장 높음. 외국인 접근성이 6개 후보 중 최상.
