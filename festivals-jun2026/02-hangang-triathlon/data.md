# 한강 쉬엄쉬엄 3종 축제 (3rd MY PACE Hangang Triathlon Festival)

| 항목 | 내용 |
|---|---|
| 행사명(국문) | 제3회 쉬엄쉬엄 한강 3종 축제 |
| 행사명(영문) | 3rd MY PACE Hangang Triathlon Festival |
| 기간 | 2026년 6월 5일(금) ~ 6월 7일(일), 09:00–19:00 |
| 장소 | 뚝섬한강공원 (Ttukseom Hangang Park) |
| 가까운 지하철역 | 확인 필요 (공식 페이지에 명시 안 됨 — 통상 경의중앙선 뚝섬한강공원역 또는 2호선 뚝섬역으로 알려져 있으나 공식 확인 필요) |
| 입장료 | KRW 30,000 |
| 외국인 접근성 | 공식 영문 페이지 운영(sium-sium.com/en), 영문 행사 소개 제공. 예약/등록 절차의 영어 지원 범위는 확인 필요 |
| 한 줄 소개 | 순위 경쟁이 아닌 "참여와 완주"가 목표인, 누구나 자기 속도로 즐기는 한강 러닝·사이클링·수영 스포츠 축제 |
| 출처 | [english.seoul.go.kr — June 2026 Cultural Events (서울시 공식 영문 사이트)](https://english.seoul.go.kr/june-2026-cultural-events/), [공식 행사 홈페이지](https://www.sium-sium.com/en/festival-overview) |
| 확인 일시 | 2026-06-28 |

> 비고: 서울시 공식 영문 월간 문화행사 캘린더(SMG 표기)에 날짜·장소·입장료·연락처(1544-5340)가 명확히 게시되어 있어 확정 행사로 분류.
