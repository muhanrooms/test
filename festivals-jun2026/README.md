## 비고 / 한계

- **지하철역 정보**: 대부분 공식 페이지에 명시되어 있지 않아 "확인 필요"로 남겨둠. 유일하게 #5(N서울타워)만 공식 페이지에 "4호선 회현역 4번 출구, 도보 500m"로 명시되어 있음.
- **외국인 접근성**: #5가 영어 통역·한영 사이트를 공식으로 명시해 외국인 대상 카드뉴스 1순위 추천 후보. #1, #3은 한국어 중심 운영으로 캡션에서 "현지 분위기 체험" 톤으로 보완 필요.
- **잠수교 뚜벅뚜벅 축제**는 4월부터 매주 일요일 운영되는 시리즈 행사라, 카드에는 "6월에는 6/7·6/14 두 번 더 열린다"로 좁혀서 소개하는 것을 권장.
- 블로그·카페 등 비공식 출처는 1차 검색 단계에서 참고만 했고, 최종 데이터는 모두 공식 출처로 교체·확정함.
