# 2026 서울시향 강변음악회 (2026 SPO Riverside Concert)

| 항목 | 내용 |
|---|---|
| 행사명(국문) | 2026 서울시향 강변음악회 |
| 행사명(영문) | 2026 SPO Riverside Concert |
| 기간 | 2026년 6월 13일(토), 19:30 (입장 18:30~), 약 90분 |
| 장소 | 여의도한강공원 물빛무대 (Mulbit Square, Yeouido Hangang Park) |
| 가까운 지하철역 | 확인 필요 (공식 페이지에 명시 안 됨 — 통상 5호선 여의나루역으로 알려져 있으나 공식 확인 필요) |
| 입장료 | 무료 (티켓 불필요, 누구나 입장 가능) |
| 외국인 접근성 | 공식 페이지 한국어 운영. 별도 영문 페이지는 확인되지 않음(서울시 공식 영문 캘린더에는 요약 정보로 영어 게재). 유튜브 실시간 스트리밍 제공 |
| 한 줄 소개 | 김선욱 지휘, 한강에서 펼쳐지는 무료 야외 클래식 공연 + 불꽃놀이가 함께하는 가족형 여름 음악회 |
| 출처 | [서울시립교향악단 공식 공연정보 페이지](https://www.seoulphil.or.kr/perf/view?perfNo=7435), [english.seoul.go.kr — June 2026 Cultural Events](https://english.seoul.go.kr/june-2026-cultural-events/) |
| 확인 일시 | 2026-06-28 |

> 비고: 공식 공연 페이지에 "No ticket is required for this annual event" / "ENTRY Free Admission(All-ages concert)"로 명확히 확정.
