# 2026 서울국악축제 (2026 Seoul Gugak Festival)

| 항목 | 내용 |
|---|---|
| 행사명(국문) | 제9회 2026 서울국악축제 — "국악, 서울을 깨우다" |
| 행사명(영문) | 2026 Seoul Gugak Festival |
| 기간 | 2026년 6월 19일(금), 13:00–22:00 |
| 장소 | 반포한강공원 달빛광장 (Central/Dalbit Plaza, Banpo Hangang Park) |
| 가까운 지하철역 | 확인 필요 (공식 페이지에 명시 안 됨 — 통상 4호선 신반포역/구반포역 또는 9호선 사평역 인근으로 알려져 있으나 공식 확인 필요) |
| 입장료 | 무료 |
| 외국인 접근성 | 서울시 공식 영문 캘린더에 영문 소개 게재. seoulgugak.com 자체는 국문 중심 — 영문 페이지 유무는 확인 필요 |
| 한 줄 소개 | 한강에서 처음 열리는 서울국악축제 — 전통과 동시대 감성이 만나는 국악 공연·체험·플리마켓 |
| 출처 | [seoulgugak.com (공식 홈페이지)](https://www.seoulgugak.com/), [english.seoul.go.kr — June 2026 Cultural Events](https://english.seoul.go.kr/june-2026-cultural-events/) |
| 확인 일시 | 2026-06-28 |

> 비고: 서울시 공식 영문 캘린더에 날짜(6/19)·장소·무료 입장·연락처(1800-4746)가 명시. 공식 홈페이지 언론보도 게시판에도 "한강에서 처음 열리는 서울국악축제"(26.06.18 게재)로 행사 확정이 재확인됨.
