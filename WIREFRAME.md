# 오늘의집(Ohou) 랜딩페이지 클론 - 와이어프레임

> 본 문서는 교육용 클론 프로젝트를 위해 https://ohou.se 메인 페이지를 분석하여 작성한 와이어프레임입니다.

## 1. 전체 레이아웃 구조

```
┌─────────────────────────────────────────────┐
│ TopBar (프로모션 띠배너)                       │
├─────────────────────────────────────────────┤
│ Header (로고 / GNB / 검색 / 유저메뉴)           │
├─────────────────────────────────────────────┤
│ SubNav (추천/집들이/집사진/커뮤니티/3D인테리어)   │
├─────────────────────────────────────────────┤
│ Hero (메인 배너 슬라이드 + 사이드 배너)          │
├─────────────────────────────────────────────┤
│ QuickMenu (아이콘 8개: 쇼핑하기 ~ 인테리어신청)  │
├─────────────────────────────────────────────┤
│ Section: 이런 사진 찾고 있나요 (가로 스크롤 카드) │
├─────────────────────────────────────────────┤
│ Section: 오늘의 추천 집들 (카드 그리드 4열)      │
├─────────────────────────────────────────────┤
│ Section: 카테고리별 상품 찾기 (아이콘 리스트)     │
├─────────────────────────────────────────────┤
│ Section: 오늘의딜 (상품 카드 4열, 가격/할인)      │
├─────────────────────────────────────────────┤
│ Section: 유저들의 인테리어 시공 리뷰 (카드 3열)   │
├─────────────────────────────────────────────┤
│ Section: 오늘의 기획전 (배너 카드 4열)           │
├─────────────────────────────────────────────┤
│ Section: 베스트 (탭 + 상품 카드 3열)             │
├─────────────────────────────────────────────┤
│ Footer (고객센터 / 회사정보 / 약관 / SNS)        │
└─────────────────────────────────────────────┘
```

## 2. 컴포넌트별 상세

### 2.1 TopBar
- 좌측: 닫기(X) 버튼 없음, 중앙 정렬 텍스트 배너
- 내용: "[ live ] 첫구매라면 누구나 최대 2만원 할인 >"
- 배경색: 브랜드 블루(#1c84ff 계열), 흰 텍스트

### 2.2 Header
- 로고 "오늘의집" (좌측)
- GNB 탭: 집구경 / 쇼핑 / 인테리어·생활
- 검색창 (placeholder: "입력해주세요", 가운데~우측)
- 우측 유저메뉴: 장바구니 아이콘 / 로그인 / 회원가입 / 고객센터 / 앱다운
- 보더 하단 라인으로 SubNav와 구분

### 2.3 SubNav
- 탭 리스트: 추천 / 집들이 / 집사진 / 커뮤니티 / 3D인테리어
- 우측: 사용자 정보 영역 "3 ▲ 집대를 들새 신반"(개인화 텍스트, 우측 끝)

### 2.4 Hero
- 좌측: 큰 메인 배너 (이미지 + 카피 문구 + 작성자 뱃지), 좌우 인디케이터 "13/16 +"
  - 카피: "인테리어 업계 종사자 부부의 현실적이지만 감각적인 구축 반셀프"
  - 작성자: "ⓚ keeeams"
- 우측: 사이드 배너 (블루 배경, "무료견적" 뱃지, "이사도 오늘의집 30초 무료견적 신청", 트럭 아이콘)

### 2.5 QuickMenu (8 아이콘)
1. 쇼핑하기
2. 오늘의딜
3. 집들이
4. 멍냥종합
5. 패키지입찰
6. 절단지찾아
7. 오마트
8. 원하는날도착
9. 이사청소
10. 인테리어신청
(아이콘 + 텍스트, 원형 배경)

### 2.6 "이런 사진 찾고 있나요?" 섹션
- 서브타이틀: "좋아하실 만한 인테리어 관련컷트 추천받아요"
- 가로 스크롤 카드 5개 (정사각 이미지 + 좌상단 태그 + 좌하단 작성자명 + 우하단 좋아요 아이콘)
- 우측 상단 "더보기" 링크
- 우측 화살표(다음) 버튼

### 2.7 "오늘의 추천 집들이" 섹션
- 카드 4열, 각 카드: 이미지 + 제목(2줄) + 본문 요약(1~2줄)
- 우측 상단 "더보기"

### 2.8 카테고리별 상품 찾기
- 아이콘 + 라벨 가로 리스트 (침구단위, 가구, 데이터, 주방·다이닝, 주방용품, 식물, 대보·소품, 초롱, 수납·정리, 생활용품, 생활용, 유아·아동, 반려동물 ...)
- 우측 화살표 버튼

### 2.9 오늘의딜
- 상품 카드 4열, 각 카드: 타이머뱃지(빨강, 시:분:초) + 이미지 + 브랜드뱃지(원형 로고) + 상품명 + 할인율(파랑) + 가격(굵게) + 별점/리뷰수 + 하단 뱃지(무료배송/적립 등)
- 우측 상단 "더보기"

### 2.10 유저들의 인테리어 시공 리뷰
- 카드 3열, 각 카드: 이미지 + "오늘의집 인테리어" 뱃지 + 본문 요약(3줄, 말줄임)
- 우측 상단 "더보기"

### 2.11 오늘의 기획전
- 카드 4열, 각 카드: 배경 이미지/컬러 + LIVE뱃지(첫번째만) + 타이틀 + 서브카피
- 우측 상단 "더보기"

### 2.12 베스트
- 카테고리 탭(전체/생활데미/가구/패브릭/가전·디지털/주방용품/식물/미모·소품/조명/수납·정리/생활용품/유아·아동/반려동물/캠핑·치즈 ...)
- 상품 카드 3열, 각 카드: 순위뱃지(1,2,3) + "오늘의집 pick" 뱃지 + 이미지 + 브랜드명 + 상품명 + 할인율 + 가격 + 별점/리뷰수 + 적립뱃지

### 2.13 Footer
- 좌측: "고객센터 >" + 전화번호 + 운영시간 + 카테고리(서비스이용, 시공문의, 제휴/입점문의, 고객센터 챗봇 등)
- 가운데: 회사정보(회사소개/이용약관/개인정보처리방침 등 링크), 사업자 정보 텍스트
- 우측: SNS 아이콘(Facebook/Instagram 등), 인증마크(ISMS, PCI 등)
- 하단: 카피라이트 "© 2026 buzzvil Inc. All rights reserved."

## 3. 컬러 / 타이포 가이드 (추정)
- Primary Blue: `#1c84ff` (포인트/뱃지/링크)
- Discount Red/Orange: `#ff5757` ~ `#ff7a00` (할인율, 타이머뱃지)
- Text Dark: `#1a1a1a`
- Text Gray: `#767676`
- Border/Divider: `#eaeaea`
- Font: 시스템 산세리프 (Pretendard / Apple SD Gothic Neo / 맑은 고딕 대체)

## 4. 인터랙션 (JS 범위)
- Hero 슬라이드 자동/수동 전환 (인디케이터 클릭 가능)
- 가로 스크롤 섹션(이런 사진 찾고 있나요, 카테고리) 좌우 화살표 클릭 스크롤
- 오늘의딜 타이머 카운트다운(00:00:00 형식)
- 베스트 섹션 카테고리 탭 클릭 시 활성 탭 전환(필터링은 더미 데이터로 시각적 전환만)
- 헤더 검색창 포커스 스타일
- 반응형: 1280px 기준 데스크탑 / 768px 이하 모바일 간소화

## 5. 에셋 처리
- 실제 이미지 대신 `picsum.photos` 또는 단색 placeholder + CSS background로 대체 (저작권 회피, 교육용)
- 아이콘은 간단한 SVG/이모지로 대체
